# Contributing to Podcast Ad Remover

Thank you for your interest in contributing to Podcast Ad Remover! This project is a "vibe-coded" personal tool that has grown into something useful for the community. We welcome contributions of all kinds.

## How to Contribute

To be clear this is a vibe-coded personal tool that has grown into something I hope can be  useful for the community. I welcome contributions of all kinds, but in all honesty I may not have the skills to actually usefully manage the feedback.

### Reporting Bugs

If you find a bug, please open an issue on GitHub. Include as much detail as possible, such as:

- Steps to reproduce the issue.
- Expected vs. actual behavior.
- Logs or screenshots (if applicable).
- Your environment (Docker, Unraid, OS, etc.).

### Suggesting Enhancements

Have a great idea? Open an issue to discuss it! We love hearing how to make the app better.

### Pull Requests

1.  **Fork the repository**.
2.  **Create a branch** for your feature or fix (`git checkout -b feature/amazing-feature`).
3.  **Commit your changes** (`git commit -m 'Add some amazing feature'`).
4.  **Push to the branch** (`git push origin feature/amazing-feature`).
5.  **Open a Pull Request**.

Please ensure your code is clean and, where possible, documented.
